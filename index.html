Independent Project

Bitcoin Mining Algorithm

As an out of class project I worked with fellow student Dylan Ayrey on creating a Bitcoin mining application made on run on an FPGA. This would allow us to run on hardware that would rival top mining machines while also being cost affective.

We built the software using Python and VHDL. We would connect to Bitcoin service on a standard linux machine. Then still on our linux box we would take the block and do all of our SHA-256 preprocessing. Then we would pass our preprocessed block to our FPGA and keep running the SHA-256 algorithm on it until we found a match to the target

School Project

Data Visualization

Working with professor Joe Pietruch I am visualizing data generated using Python's NLTK module and play scripts to show and analyze how characters interact with each other and how they evolve as a story transitions from start to finish.

I use technologies like HTML5 Canvas and Websockets to dynamically generate the visualizations through a web browser to clients all over the world.

Work Project

Planning Poker

Working with Siteworx I helped develop a software planning app which hooked directly into our JIRA servers to take stories and then assign time estimates (or "story points") on how long it will take to complete them.

Using HTML5 Websockets we allow users to create rooms and host games. Once a user has created a room they are officially a moderator for that game and can add, delete, and assign points to stories.
