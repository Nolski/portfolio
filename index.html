Independent Project

Bitcoin Mining Algorithm

As an out of class project I worked with fellow student Dylan Ayrey on creating a Bitcoin mining application made on run on an FPGA. This would allow us to run on hardware that would rival top mining machines while also being cost affective.

We buit the software using Python and VHDL. We would connect to Bitcoin service on a standard linux machine. Then still on our linux box we would take the block and do all of our SHA-256 preprocessing. Then we would pass our preprocessed block to our FPGA and keep running the SHA-256 algorithm on it until we found a match to the target
